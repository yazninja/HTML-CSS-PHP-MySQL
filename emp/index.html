The Evolution of the World Wide Web

A short introduction about static, dynamic, and sematic web.

Static Website

Also known as Web 1.0, this were the pages present when the WWW was still in its first stages. These pages are "as is" and thus cannot be manipulated by the user. The content is same for all users.

Dynamic Website

Also known as Web 2.0, It is the evolution of Web 1.0, by add DYNAMIC webpages. By using dynamic webpages the user is able to see a website differently than other users. it lso allows users to use web browswers as user interfaces or application softwares. Most Website nowadays are coded in Web 2.0

Web 3.0

This was coined by the inventor of WWW himself. Tim Berners-Lee. The aim of web 3.0 is to provide users with more personalized information based on the server's understanding of the user preferences. Alhough this cannot be implemented nowadays, we could expect this in the following 10 - 20 years, through the advent of technologies.

A website about, well, Aaron Porter. It's a website that talks about his personal information and works. Every visitor to this site will get the same information as the other as there is no user interaction in this website.

Aaron Porter Website

AT

Atlassian Infographic Page

This is an example of static website because this website just gives you information and the data is fixed and cannot be change.

Atlassian Infographic Website

ME

Metalsmith

is a pluggable static website generator. It gives tutorials and information on how to create your own static website through given plug-ins and instructions. It is also, by itself, a static website.

Metalsmith Website

HU

Hugo

Hugo is a static site generator written in Go. It is one of the most popular open-source static site generators. All of their pages ar coded in HTML and displays the same information to every visitor.

Hugo Website

OS

Orange Sprocket

OrangeSprocket is a Creative Technology Company focusing on digital branding, user experience (UX / UI), creative design, mobile and web application design. It is considered as a static website for the data is fixed and doesn't change unless it has been added in the html code. It is a web page that is delivered to the user's web browser exactly as stored, in contrast to dynamic web pages which are generated by a web application.

OrangeSprocket Website

AM

Amazon

Amazon is an example of a dynamic website since it is an e-commerce website. It can be customized and easily be manipulated by the user.

Amazon Website

PA

Paladins

This is an example of a dynamic website or Web 2.0 for it displays different content each time it is viewed. It can be changed through the user that accesses the webpage, or the type of user interaction.

Paladins Website
